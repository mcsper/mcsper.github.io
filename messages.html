Sweet! That's a good sign if you can see the config. Do you have anything in your log in the same directory? My guess is that you will.

To access the HA frontend it should be as easy as accessing your Pi's internal IP address at port 8123. So something like 192.168.0.100:8123

You need to make sure both devices are on the same network. So, if your Pi is connected to a wired port a wireless router, your PC should be too. Typically, you can't access HA on the "backside" of a switch or router without some kind of port forwarding configuration.

My guess is if you can SSH into the Pi, you should be able to access HA on the same IP address from that device. If it's not the network then HA might be crashing on boot. Check the log to see if there is an error statement. A reinstall might fix it, too.

The first time you fire up HA, it can take like 45+ minutes for it to initiate the frontend. It has to download a lot of dependencies... especially on a Pi1.

If you get it up and running, I'd definitely recommend switching to a Pi3. It's way way faster and will definitely improve your loading times and such. Not to mention, you can never have enough Pis.

Let me know if that doesn't help. I can try and think of some more things, as well. Good luck! Cheers!
